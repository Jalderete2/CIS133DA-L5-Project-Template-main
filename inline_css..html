Generic vs Specific Fonts

Generic font families define a broad category of fonts such as serif, sans-serif, or monospace. Specific fonts are individual typefaces within those categories, like Arial or Times New Roman. Using generic families ensures text remains readable if a specific fontbis unavailable.
